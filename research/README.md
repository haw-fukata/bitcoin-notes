# Research notes

公開前の調査メモを置く場所です。テーマに合うディレクトリを選び、`templates/research-note.md` をコピーして使います。

| ディレクトリ | 対象 |
| --- | --- |
| `protocol/` | トランザクション、UTXO、スクリプト、P2P、コンセンサス |
| `mining/` | Proof of Work、難易度調整、ブロック生成、マイニング市場 |
| `lightning/` | ペイメントチャネル、ルーティング、各種実装 |
| `economics/` | 発行量、手数料市場、ゲーム理論、金融・経済 |
| `security/` | 脅威モデル、鍵管理、攻撃、脆弱性 |
| `history/` | Bitcoinの歴史、人物、出来事、ソフトフォーク |

複数分野にまたがる場合は、中心となる問いに最も近い場所へ置き、関連メモからリンクします。
